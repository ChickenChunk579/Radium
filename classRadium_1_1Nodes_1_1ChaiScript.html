Detailed Description

Script implementation using ChaiScript for scripting support.

Loads, manages, and executes ChaiScript scripts attached to scene nodes.

Constructs a ChaiScript instance.

Loads the script from the specified file path and attaches it to the given scene tree. Optionally, the loading can be stubbed (not fully loaded) for editor or preview purposes.

Parameters
pathPath to the ChaiScript script file.
treePointer to the SceneTree the script will be attached to.
realLoadWhether to perform a real script load (default is true).
